Textpad Editor

Textpad Editor is a simple, general-purpose, cross-platform text editor written using the Qt toolkit. Textpad aims to be faster and more lightweight than most text editors, but as useful as them. Additionally, Textpad supports syntax highlighting, word count, line count and comes with 5 different color schemes.

Features

  • Simple and clean interface.
  • Has 5 different color schemes.
  • Distraction free features (such as full screen and/or Minimal Mode).
  • Includes "Minimal Mode", where the toolbar and statusbar are hidden.
  • Supports more advanced features, such as syntax highlighting and line numbers.
